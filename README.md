## 🧑‍💻 Who I Am

> *"I care about both sides of the stack — clean APIs are just as important as clean interfaces."*

I'm an **aspiring full-stack developer** who started with UI — obsessively recreating real-world products like Spotify and Notion until every pixel matched — and is now building the complete picture: **frontend + backend + database.**

I'm actively working with **React, Node.js, Express, and MongoDB** to ship end-to-end applications. My UI background means the apps I build don't just work — they look and feel right too.

- 🎯 **Current focus:** Full-stack MERN apps — auth, REST APIs, dynamic UIs  
- 📦 **Side project:** Portfolio site built from scratch — no templates, no shortcuts  
- 💡 **Learning style:** Build first, break it, understand why, repeat  
- 🤝 **Open to:** Full-stack internships, freelance work, open-source collabs
